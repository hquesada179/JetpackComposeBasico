# JetpackComposeBasico

Proyecto básico en **Jetpack Compose**.

## Descripción
App simple que muestra una pantalla de perfil con imagen y datos básicos usando Compose.

## Tecnologías
- Kotlin
- Jetpack Compose
- Android Studio

## Ejecución
1. Abrir el proyecto en Android Studio
2. Esperar sincronización de Gradle
3. Run ▶️ en emulador o dispositivo
